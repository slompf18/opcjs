# User Token – Anonymous Client

**Specification**: OPC 10000-7 §9.x (Profiles), version 1.05  
**Profile URI**: `http://opcfoundation.org/UA-Profile/Security/UserToken/Anonymous`  
**Category**: Security  

## Overview

The User Token – Anonymous Client facet defines the minimal user authentication requirement: the client must be able to connect as an anonymous (unauthenticated) user.
