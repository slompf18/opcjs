# User Token – Anonymous Server Facet

**Specification**: OPC 10000-7 §6.x (Profiles), version 1.05  
**Profile URI**: `http://opcfoundation.org/UA-Profile/Security/UserToken/Server/Anonymous`  
**Category**: Security  

## Overview

This Facet indicates that the Server supports anonymous User Tokens.
Servers must allow administrators to disable or enable this token type — it should be disabled in production environments where unauthenticated access is not permitted.
